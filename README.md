# Worker Buddy

## Setup
pip install -r requirements.txt
playwright install chromium

## Run
python main.py

## Usage
- Type a plain-English task in the text box
- Click RUN
- Watch the agent work in the log area
- Right-click the system tray icon (indigo dot, bottom-right taskbar) for settings

## Notes
- Credentials loaded from C:\JSON Credentials\QB_WC_credentials.json
- Window can be dragged from the title bar or log area
- Drag the bottom-right corner to resize
- Minimize hides to tray; the agent keeps running
